# Implementation-of-Kruskal-s-and-Prim-s-Algorithms-for-Minimum-Spanning-Tree
A Python and Streamlit application that compares Kruskal's and Prim's Minimum Spanning Tree (MST) algorithms by generating the MST, calculating the total cost, and analyzing their performance on weighted graphs.

## Overview
This project compares two popular Minimum Spanning Tree (MST) algorithms:
Kruskal's Algorithm,
Prim's Algorithm.

The application generates the Minimum Spanning Tree (MST) of a weighted graph and compares:
MST Edges,
Total Cost,
Algorithm Performance.

## Technologies Used
Python,
Streamlit.

## Algorithms
Kruskal's Algorithm,
Prim's Algorithm.

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
